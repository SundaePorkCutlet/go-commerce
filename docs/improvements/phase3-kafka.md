## 개선 전 문제점

| 문제 | 설명 |
|------|------|
| PRODUCTFC 컨슈머 없음 | stock.updated/rollback 이벤트를 발행하지만 소비하는 곳이 없음 |
| DLQ 없음 | 처리 실패 메시지가 log 찍히고 사라짐 |
| 멱등성 미보장 | 같은 메시지를 2번 받으면 2번 처리됨 |
| 파티션 전략 없음 | LeastBytes 밸런서로 순서 보장 불가 |
