---

## 개선 전 문제점

| 문제 | 설명 |
|------|------|
| 커스텀 메트릭 0개 | /metrics 엔드포인트만 열어두고 등록된 메트릭 없음 |
| 대시보드 없음 | Grafana를 띄워놨지만 대시보드 JSON 없음 |
| 알림 없음 | 장애 발생해도 알 방법 없음 |
| 트레이싱 미완성 | HTTP 미들웨어 스팬만, Kafka/gRPC 트레이싱 없음 |
